
# Algorizo ARC ERP 팀 프로젝트

# 주요 기능
- 공지사항
- 로그인 / 회원가입
- 사원 관리
- 품목 관리
- 공급 업체 관리
- 조달 계획 관리
- 발주 관리
- 계약 관리
- 입고 / 출고 관리
- 입고 / 출고 거래명세서
- 입고 / 출고 검수
- 재고 관리

# 기술 스택
- 언어 : Java 11
- 프레임 워크 : Spring Framework (STS3 + Maven)
- Front : JSP, JSTL, Bootstrap
- ORM : MyBatis
- BD : MySQL
- 보안 : BCrypt
- 서버 : Apache Tomcat 9
- 형상관리 Tool : Git(SourceTree)
